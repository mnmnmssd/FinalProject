# 期末项目-----个人博客系统

学校期末大作业，基于`SpringBoot`+`mybatis`+`redis`的个人博客系统

基于老师原本项目基础上进行扩展，主要扩展以下功能

![image-20211019211243949](https://cdn.jsdelivr.net/gh/mnmnmssd/hexoBlogimg/blog/2020/image-20211019211243949.png)

具体实现了以下功能:

- 用户登录
- 权限控制
- 文章发布、修改、删除
- 评论发布、删除
- 文章分类管理
- 用户管理
